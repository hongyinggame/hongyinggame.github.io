SQL,易忘命令

  SQL使用;作为结束标志，SQL不区分大小写，但是一般关键字大写是为了更容易阅读，SQL查询结果没特殊说明，不进行排序，SQL对空格不进行处理，所以可以写在一行，也可写在多行。

SELECT DISTINCT 列名 FROM 表名; DISTINCT 用于查询不同字段，消除重复字段。
SELECT TOP 5 列名 FROM 表名; TOP 用于指定前几行记录;
MYSQL使用如下 SELECT 列名 FROM 表名 LIMIT 5 OFFSET 5; LIMIT关键词用于指定输出几行，OFFSET 用于指定开始行号，可简化为LIMIT 5,5; 但是第一个是OFFSET值，第二个为LIMIT值

《伞少女》

  《伞少女》这部电影，我的感觉是音乐与画面不错，真的是展示汉文化，但是剧情会欠缺，然后我之前看了《百妖谱》，就感觉煽情有点难以接受了，这是次要的，主要是感觉剧情有点赶，铺垫不是特别好，导致情不到位啊，而且其实有些强行煽情的感觉，其实，我感觉有时无言往往胜于有言，有时需要一些无言的动作，而非全都融于言语，这样会好很多，最后情节会好很多了，前面真的有点铺垫不是好，总之，还不错，挺喜欢黑玉剑的。
  独立自主，要能够以自己的意志去表达，去执行，成为一个真正的人。
  人与物同时存在好坏善恶，关键在于我们能否合理运用，能否看见其本质，能否看到两面性，同时，每个人与物都有其存在的意义与价值，无论过程怎样，结果怎样，生存于世就一定有其意义，各种意义上的。
  物存在本无好坏，或者说可为坏也可为善，重点不在于物，而在于使用者，在于人，所以那些说物是恶的，物为万恶之始，物降世或现世，就会带来祸运的，都是错的，大错特错，物本没错，错在于使用者，是否能够利用好物，可以造福也可以招祸，比如剑，一般认为是杀戮，所以不想让他现世，但是有没有想过，很多时候一些坚硬物体是需要剑或刀来工作的，保护不仅仅是防御，还有进攻，伞能保人的安全，但无法根除祸害，说到底还是没有保护好人，因为它只能保护眼前之人，但是剑可以铲除祸害，从而保证后世不会有这个祸害，因此，从这个意义上来说，剑不是只有杀戮，还有守护，最主要在于使用者，而非器物。
  不经人之苦，莫劝别人善。如果我们不理解别人，不能真正体会别人，那么我们就不应该将自己的意志强加于别人，因为有时这只会让别人更痛苦，所以说教还是肃杀，是应该合理抉择，不能强词说教，也不能滥杀。
  人一旦为了物灵献出生命，物灵也就能够变成人的样子。这句话不能理解错误，这句话不是说人为物灵牺牲，物灵就可以变成人了，而是应该明白，人的牺牲需要让物灵觉醒自我意识，需要物灵能够遵从自己的愿望，形成自我意志，自由或许就是能够以自我意志来活着，能够依据自我愿望生存。
  善良不是表演，不是过度展示，善是朴实无华，是无须雕琢，暖人心而非悦人眼，无言之善才是真正的发自内心之善。
  无心之作，但被人过度解读之后，便是有心之作。很多时候，人的无心之举，对别人没有任何意义，但是往往大道隐含与平常之事、平常之物中，往往我们看到很多无心之事，也会蕴含一些本质，关键在于人的眼睛，能否看透表象隐含的本质。
  电影无论什么类型只要是人制作的，其都有含义，都有所要表达的东西，因为人是有感情的。
  他守天下，我守他，器以自己的意志强加于人，但他却坚决自刎。

动态-24年7月5日

  今天心情比较我沉重，感觉人生没有意义，所以想着去放松一下心情，然后发现，可能人或者确实没有意义，最后都是死，活着一生到底为什么呢，不清楚，但是我认为，虽然人生确实活着没有意义，但是世界那么多彩，那么多样，所以，我活着不是为了人生意义，我活着是为了能够看到更多、更丰富的事情，能够看看我们这些存在于世的人所创造出的事物，无论好坏，还是善恶，那么多的事情我都没有看到，太亏了，我要去看看，这就是我的人生意义。
  最后，我要努力去活直到生命终止，然后，尽可能去看看世界，去看看我们这些人所创造的事物，去感受我们所创造的世界。
  科学可以救人，可以杀人，可以欺骗，也可以诚实，那些不同科学的人其实就是科学的奴隶，科学想怎么弄他们，就怎么弄他们，所以真的很可悲，最后科技竟然是玩弄于人，而不是服务于人，因为科学是掌握在人手中，言语权也是，所以也就有善有恶，所以我们要掌握核心科技，不能成为那些懂科技人的奴隶，被他们玩弄于股掌之中。

《默杀》

  今天看了《默杀》电影，整体我认为效果很好，他把恐怖元素运用的很好，环境、氛围以及声音和jump scare，真的有好几个画面够让我吓一跳的，如果电影院就我一个，我可能会很害怕，还好不是，扯远了，除了这个，我认为剧情也不错，紧凑不拖拉，每个环节都有伏笔，虽然我能够看到，但是无法深入推断，或者说推断错误，或者说这根本不是推断，而是猜想而已，又扯远了，整体就是观感不错，有几个画面，演员表演的也很不错，尤其第一个虐待女主时的情节，那个女孩的眼神很有感觉，总之很不错的一部电影，在此我要说一句：我真的恨死jump scare了，虽然明知道会有，但是因为你不知道何时以及你不知道是什么形式的jump scare，所以无论如何都会吓一跳，太恶心了，人就是会对未知好奇与恐惧，会让人臆想连篇，会让人情不自禁地去想，自己吓自己。
  然后，就是整部电影都体现一个默，校长女孩杀人后沉默，女主与女主妈沉默于杀害继父，女主沉默于整个计划，女主朋友他爸杀人后也沉默，整个都是沉默，沉默着死去，沉默着活着，真的是不在沉默中爆发，就在沉默中灭亡，最后女主爆发了，进行了复仇，然后解脱。
  其实，我很大程度认为，这一切都是那个女孩谋划的，借刀杀人，让她母亲作为她的替罪人，但是她不知道，那个偷窥狂有真正的杀人证据，但是她妈妈为了保护她，选择了袒护，她的哑巴可能是装的，用于躲避杀人事实，躲避与人交流，但是她妈妈知道吗？真的不清楚。
  最后，水只有到达一百度才会迸发，人是往往会忍耐，但是忍耐是有限度的，我们应该控制好度，要避免水沸腾而流出外面来，不要得寸进尺，更不要挑战底线，人要么沉默中死亡，要么沉默中爆发，而爆发往往是被逼压而造成的，压强大了就包不住了。

头脑特工队1

  随着长大，我们会忘记一些我们曾经的记忆，但是，我们要明白，我们要感谢那些好的记忆，因为我们就是从他们中产生的，有些小时候的习惯，大了也变淡了，但是，熟不知那些习惯，可能为我们打下了良好的基础，其实，我也认为有些好的习惯不能忘记，更不能抛弃，比如我至今都保持着想象自己为主角的故事，因为这能够培养我的想象力与大脑图像化能力，以及我的记忆力，我认为我目前的能力，很大部分都是由它培养的，所以我不会抛弃这个，虽然最开始，这个习惯是为了让我能够自娱自乐，能够让自己更快乐,虽然略显中二，但是这是我至今为止一直保持的，对我来说最重要的能力。
  然后，有些记忆，在最开始时快乐的，但是我们不能将其强制在这个情绪，有些时候，我们需要以另一个情绪去分析观察，或者说以其对立面去看待，因为很多时候情绪是共生与共存的，用另一个情绪观察会有不一样的感觉，也许有可能产生另一种意义上的快乐，总之，学会包容、开放，不要极端化以及固化，要灵活与适度平衡，放手自己，勇敢面对，你会发现生活其实很有意思的。
